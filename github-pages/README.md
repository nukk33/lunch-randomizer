# GitHub Pages Ready

ไฟล์สำหรับอัป GitHub Pages อยู่ในโฟลเดอร์นี้

## ใช้งาน

1. สร้าง GitHub repository ใหม่
2. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้นไปที่ root ของ repository
3. ไปที่ Settings > Pages
4. ตั้งค่า Source เป็น `Deploy from a branch`
5. เลือก branch ที่ใช้เก็บไฟล์ และเลือกโฟลเดอร์ `/ (root)`
6. กด Save

หน้าเว็บจะใช้ไฟล์ `index.html` เป็นหน้าหลักอัตโนมัติ
